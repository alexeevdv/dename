dename - Desktop Environment NAME
======

Bash script to detect linux desktop environment name and version

How to use:

* `wget https://raw.github.com/alexeevdv/dename/master/dename.sh`
* `chmod +x dename.sh`
* `./dename.sh`

Can detect:

* Gnome 2, Gnome 3
* KDE4
* Ubuntu Unity
* XFCE
* Cinnamon
* Mate
* LXDE

Tested in:

* Knoppix 7.0.5
* Lubuntu 13.10
